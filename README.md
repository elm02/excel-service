# Сервис поиска N-ного минимального числа из Excel файла

Spring Boot сервис для поиска N-ного минимального числа в Excel файле

## Технологии
- Java 17
- Spring Boot 3.5.7
- Apache POI 5.4.1
- SpringDoc OpenAPI 2.2.0 

## Возможности
- Чтение целых чисел из .xlsx файлов
- Поиск N-ного минимального числа 
- Swagger UI для тестирования API

## API Endpoints
`GET /api/excel/find-smallest`
##### Параметры:
- **filePath** - путь к локальному .xlsx файлу

- **n** - порядок минимального числа
##### Пример:
`GET http://localhost:8080/api/excel/find-smallest?filePath=D:/numbers.xlsx&n=3`

## Алгоритм поиска

#### Принцип работы:

1. **Выбор опорного элемента** - выбирается последний элемент массива
2. **Разделение массива** - массив разделяется так, что все элементы меньше опорного перемещаются в левую от него часть, а большие - в правую.
3. **Итеративный поиск** - если элемент не найден, алгоритм повторяется с той половиной, где находится требуемый индекс

#### Преимущества алгоритма:
- Не требуется полная сортировка массива
- Алгоритм работает с исходным массивом и не требует дополнительной памяти

## Запуск проекта

#### 1. Сборка проекта через Maven

Для сборки проекта используется команда

`mvn clean compile`

#### 2. Запуск проекта

Для запуска проекта используется команда

`mvn spring-boot:run`

## Доступ к Swagger UI
После запуска сервиса документация API доступна по адресу:

`http://localhost:8080/swagger-ui.html`

### Формат файла

Файл должен соответствовать требованиям:

- Формат .xlsx (Excel)
- Числа расположены в первом столбце
- Поддерживаются только целые числа
- Пустые ячейки игнорируются
